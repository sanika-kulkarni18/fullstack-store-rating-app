This is a full stack web application built to allow users to view stores, give ratings, and write reviews. It also includes different roles like Admin, Store Owner, and Normal Users with proper access control.

The idea of this project is to make a simple system where users can share feedback about stores and store owners can see how their store is performing.

Features:

 Users can register and log in to the system
 Browse and view available stores
 Give ratings and write reviews
 Store Owners can view ratings and feedback for their own store
 Admin can add, update, or delete users
 Manage store listings
 View all ratings and reviews

Tech Stack Used
Frontend:
HTML, CSS, JavaScript 
Backend:
Node.js with Express.js
Database:
MySQL
